Simple Keyboard Layout Icon
----

Tiny icon which shows the current keyboard layout as icon in system tray.

# Setup

make
make install

# Uninstall

make uninstall

# Usage

Help:
simple_xkb_tray_icon --help

Silent output:
simple_xkb_tray_icon --silent

Regular output:
simple_xkb_tray_icon
